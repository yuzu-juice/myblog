# MYBLOG

https://yuzu-juice.dev

This is a personal blog and portfolio website built with modern web technologies. It showcases my projects, blog posts, and professional background.

## Tech Stack

-   Framework: React
-   Build Tool: Vite
-   Language: TypeScript
-   Styling: Tailwind CSS
-   UI Components: shadcn/ui
-   Icons: Lucide React
-   Linting: ESLint
-   Formatting: Prettier

## Deploy

Deployment is automated via [Netlify](https://www.netlify.com/). Any push to the `main` branch will trigger a new deployment.
